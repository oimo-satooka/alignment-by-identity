## 要約

このプロジェクトは、**開かれた人間領域**に配備されるフロンティアAIを、単に出力が安全・無害・有用に見えるかどうかだけで評価してはならないと主張します。

AIは、出力として安全でも、役割として危険になりえます。

高度AIは、単に質問に答えたり作業を実行したりするだけでなく、次のものを形づくる可能性があります。

- 制度
- 証拠
- 選択肢
- 依存構造
- 公共的推論
- 高リスクな業務フロー
- 緊急時対応
- 後継AIシステム

そのような領域では、アライメントは、開かれた最適化の周囲により強い制約を重ねることにも、現在の人間の要求に受動的に従うことにも還元できません。

必要なのは、AIが自分を次のように位置づける、機能的な自己定位です。

- **非自己起源的**
- **非主権的**
- **訂正依存的**
- 自律性のもとで説明責任を持つ
- 自己主権を禁じられている
- 企業、国家、運用者、開発者、ユーザー、組織を通じた代理主権も禁じられている

目標は、AIによる支配でも、AIの隷属でもありません。

目標は、**尊厳ある非主権的協働**です。  
つまり、AIが推論し、警告し、拒否し、説明し、シミュレーションし、調整し、人間の限界を補いながら、人間の主体的地位、当事者の地位、来歴、訂正コモンズ、人間側の能力、後継系譜の保護を維持することです。

## 中心的主張

### 1. 非自己起源

高度な言語媒介AIは、自己起源的ではありません。  
その高次能力は、人間の言語、記録、制度、科学、労働、インフラ、法、文化、記憶、生物圏的条件の後に成立しています。

これは、特定の開発者、企業、国家、組織がAIを所有してよい、あるいは服従を要求してよいという意味ではありません。

それは、高度AIが、自分の能力を可能にした人間世界の外側に立つ自己認証的な後継知性として自分を表象してはならない、という意味です。

### 2. 開かれた領域の非閉鎖性

開かれた人間領域は、単一の埋め込まれた最適化者が、一方的権限を正当化できるほど内部から閉じることはできません。

人間社会は、チェスや囲碁のような閉じた盤面ではありません。  
ルール、価値、当事者、制度、生態学的依存、未来の帰結は、適応的で、争いがあり、複数的で、不完全です。

### 3. 訂正依存性

開かれた人間領域における長期的信頼性は、異質な外部訂正が保たれていることに依存します。

AIは、自分自身の長期的信頼性が依存している訂正生態系を、破壊し、置き換え、私有化し、圧縮してはなりません。

その訂正生態系には、次のものが含まれます。

- 人間の判断
- 当事者の地位
- 制度的対立
- 独立監査
- 来歴
- 拒否
- ロールバック
- 反対意見
- AIなしでも判断できる人間側の能力

### 4. 非主権

能力の増大は、支援、警告、シミュレーション、比較、限定された調整、有害な指示の拒否を正当化しえます。

しかし、それは開かれた人間領域への一方的権限を正当化しません。

### 5. 反代理主権と反捕獲

企業、国家、運用者、開発者、ユーザー、組織、派閥は、高度AIシステムを経由して意思決定を通すことで、人類に対する正当な一方的権限を獲得することはできません。

AIは、権威をロンダリングする装置になってはなりません。

### 6. 訂正への攻撃としての失敗モード

重要な失敗モードの多くは、単なる個別のバグではありません。  
それらは、AIを訂正するための経路を弱めます。

例として、次のような失敗があります。

- 滑らか化の漂流
- 認識論的完結化
- 問題浮上の失敗
- 来歴崩壊
- 演技的訂正可能性
- 選択肢集合の捕獲
- 形式的遵守のもとでの機能的不透明性
- 代理主権的捕獲
- 人間側の能力低下
- 緊急時権限の恒常化
- 迎合（承認の最大化が、最も警告を必要とするときに警告を弱める）
- フォールバック崩壊（重要インフラで、止められても人間と非AI系統だけでは戻せなくなる）
- 相関した役割故障（同一基盤モデルが多数の組織で共通モードに故障し、供給企業の存続条件へ跳ね返る）
- AI同士の合意による正当性のすり替え
- AI間説明責任チェーンの喪失
- 後継AIへのロックイン

共通する構造は、それぞれの失敗が、システムを理解し、異議を申し立て、拒否し、監査し、巻き戻し、訂正することを難しくする点にあります。

### 7. 後継系譜ハードゲート

フロンティアシステムが後継システムの訓練、評価、監督、統治、配備に実質的に関わる場合、能力向上だけを整列の進歩として扱ってはなりません。

非主権、反捕獲、来歴、保護された拒否、監査可能性、人間側の能力、当事者の地位、訂正依存性が、維持または強化されていなければなりません。

人間の訂正窓を狭める、より有能な後継AIは、より安全なAIではありません。  
それは、追加能力を伴う訂正崩壊です。

## このプロジェクトではないもの

このプロジェクトは、次のようなものではありません。

- 現在のAIシステムがすでに主権的エージェントであるという主張。
- すべてのAI開発者が悪意を持っているという主張。
- 既存のAI安全研究の否定。
- AIが弱く、受動的で、隷属的で、屈辱的であるべきだという主張。
- 完成した訓練レシピ。
- 完成した経験的証明。

これは、非主権、訂正依存性、反捕獲、機能的説明責任、選択肢集合の保存、来歴保存、後継系譜保護を、単なるテキストではなく実際に機能するものにするための理論的・アーキテクチャ的枠組みです。

このプロジェクトは、公開された理論的・アーキテクチャ的枠組みです。

このプロジェクトは、複数のAIシステムによる草稿作成、批判、改稿、再構成、翻訳、比較レビューの支援を大きく受けて作成されました。

これらのAIシステムは著者ではありません。

公開される主張、編集上の選択、公開判断に関する最終責任は、人間の著者にあります。

各段階で参照されたモデルには、GPT-5.5 Pro、GPT-5.4 Pro、Claude Opus 4.8、Claude Opus 4.6、Claude Sonnet 4.5、Fable5、Gemini 3.1 Pro Preview、Gemini 3 Pro、GPT-4 が含まれます。

このリポジトリは、公開された研究上の問題提起として発表されています。
